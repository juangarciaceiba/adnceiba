# ADN Ceiba - CRUD Personas

## Descripción
API REST para gestión de personas desarrollada con Spring Boot como parte del programa de entrenamiento ADN Ceiba.

## Tecnologías
- Java 17
- Spring Boot
- Maven
- PostgreSQL

## Estructura del Proyecto



## Cómo Ejecutar
1. Clonar el repositorio
2. Ejecutar: `mvn spring-boot:run`
3. La API estará disponible en `http://localhost:8080`

## Endpoints Disponibles
| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | /api/health | Health check de la API |
| GET | /api/welcome/{nombre} | Saludo personalizado |
